## Setup

- Clone the project
- `npm install`
- Copy `.env.example` to `.env`
- `npm run db:migrations`
- (optional) Seed the database with `npm run db-seed`
- `npm run dev` to start the server

## Reset the DB

- Delete ./data.db
- Delete the contents of app/db/migrations
- Run `npm run db:migrations`
- (optional) Seed the database with `npm run db-seed`
